Sync ke Google Sheets

Sistem hantar data artikel terus ke Google Sheets anda melalui Google Apps Script (percuma, tiada API key diperlukan).
1. Setup
2. URL
3. Sync!
Langkah 1 — Setup Google Apps Script
1. Buka sheets.google.com → cipta spreadsheet baharu
2. Namakan: Pangkalan Literatur Tesis
3. Klik Extensions → Apps Script
4. Padam semua teks sedia ada dalam editor
5. Salin kod di bawah → tampal → klik Save
6. Klik Deploy → New deployment
7. Type: Web app → Execute as: Me → Who has access: Anyone
8. Klik Deploy → Authorize access → salin URL yang dijana
Langkah 2 — Tampal URL Web App
Selepas Deploy, Google akan beri URL seperti:
https://script.google.com/macros/s/AKfycbx.../exec
URL ini disimpan dalam storan peribadi perbualan ini sahaja — tidak dikongsi.
